# 3. Key Decisions

### Grain Preservation
- Confirmed that **`collision_index`** is the primary key for the parent **collisions** dataset.
- The field must remain a **string** to preserve leading zeros and maintain referential integrity.

### Storage Boundary
- Stored only **small sample datasets (~1,000 rows)** in the GitHub repository under:
  ```
  data_sample/raw/
  ```
- Stored the complete datasets in **Databricks Volumes** to comply with project guidelines and repository size limits.

### Reproducibility
- Fixed the synthetic data generator with:
  ```python
  RANDOM_SEED = 42
  ```
- Ensures identical outputs every time the script is executed, making results reproducible for reviewers.
